# Laravel Commands

> Laravel Artisan Command Utilities v1.0.12

## Screen

![Screen](https://raw.githubusercontent.com/marceloxp/laravel_commands/master/screen-v3.png "Screen")

## Install

```bash
composer require marceloxp/laravel_commands
```

## Usage

```bash
php artisan xp
```

## Commands

### Migrate

- Status
- Create Model
- Create custom migrate
- Preview
- Rollback
- Execute migration

### Seeds

- Create
- Execute One
- Execute All

### Composer

- Dump autoload

### Database

- Show config (from .env)
- Show tables
- Show fields from table
- Rules generator
- Dump database
